# Push Animation

`MenuComponent` pushes the `App` outside the viewport.

## Note

You need to add `overflow: hidden` to the root of your `App`, otherwise you will get a horizontal scroll when the `MenuComponent` is open.

For `create-react-app` you can do:

```
#root {
  overflow: hidden;
}
```
