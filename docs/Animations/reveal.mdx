# Reveal Animation

`App` slides outside the viewport revealing the `MenuComponent`.

## Note

You need to add `overflow: hidden` to the root of your `App`, otherwise you will get a horizontal scroll when the `MenuComponent` is open.

For `create-react-app` you can do:

```
#root {
  overflow: hidden;
}
```

Also, make sure that your `App` has a solid background & is covering the entire viewport height otherwise you won't see a reveal effect as `App` is not hiding the `MenuComponent` in the first place.
